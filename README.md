
# AI Placement Mentor

An AI-powered platform that helps students improve their resumes, prepare for interviews, identify skill gaps, and receive personalized career guidance.

## Features (Planned)

- Resume Analysis
- ATS Score
- Job Description Matching
- Skill Gap Analysis
- AI Interview Coach
- Personalized Learning Roadmap
- Cover Letter Generator

## Tech Stack

- Python
- Streamlit
- Gemini API
- PyMuPDF
- Pandas
